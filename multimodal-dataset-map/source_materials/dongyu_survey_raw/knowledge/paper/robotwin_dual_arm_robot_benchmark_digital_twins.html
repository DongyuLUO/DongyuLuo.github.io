RoboTwin: Dual-Arm Robot Benchmark with Generative Digital Twins

Mu, Chen, Chen, Peng, Lan, Gao, Liang, Yu, Zou, Xu, Lin, Xie, Ding, Luo · 2025 · arXiv preprint · arXiv:2504.13059 · PDF

Dongyu supplement. Added as a candidate missing/adjacent dataset paper after searching primary arXiv sources. This is a draft summary for triage, not a full paper read.

One-liner. A dual-arm manipulation benchmark that uses generative digital twins and LLM-assisted code generation to create diverse expert data and real-world-aligned evaluation.

Setup

  • Datasets / benchmarks: Introduces RoboTwin: a benchmark with simulated and real-world data for dual-arm tasks. It creates object digital twins from single images and uses large language models to decompose tasks and generate movement code.
  • Hardware / simulator: Validated with the open-source COBOT Magic robot platform; combines simulated and limited real-world data.

Method

Generative digital twins plus spatial-relation-aware LLM code generation for task decomposition and expert data generation.

Why it matters for the map

Useful for the benchmark-design side of the project, especially around language-generated task structure and sim-to-real evaluation.

Limitations / open questions

Not primarily a tactile/audio/force/thermal dataset; language is used for task generation rather than sensor-grounded annotation.

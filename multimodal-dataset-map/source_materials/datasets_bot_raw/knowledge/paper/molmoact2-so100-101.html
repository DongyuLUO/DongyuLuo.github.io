MolmoAct2 SO-100/SO-101 Dataset

Allen Institute for AI (Ai2) · 2026 · datasets.bot · datasets.bot page

One-liner. Ai2 language re-annotations for 1,220 community SO-100/SO-101 LeRobot manipulation datasets.

Setup

  • Datasets / benchmarks: A MolmoAct2 resource from Ai2 providing per-episode annotated language instructions for low-cost SO-100 and SO-101 arm data sourced from 1,220 community LeRobot repositories (377 users). It does not re-host trajectories; each source dataset gets a tasks_annotated.parquet of refined instructions indexed by episode, with the source repos listed in repo_list.json. License: Apache-2.0. Download: https://huggingface.co/datasets/allenai/MolmoAct2-SO100_101-Dataset.
  • Hardware / simulator: Embodiment: parallel_jaw. Environment: tabletop. Realness: physical.

Schema

Manifest of 1,220 source SO-100/SO-101 LeRobot repos + per-repo tasks_annotated.parquet of refined per-episode language instructions; repo_list.json lists originals.
